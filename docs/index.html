# jason-helper

项目中积攒下来的实用 js 工具库
宗旨为只包含 es api，所以 node，小程序中也可以使用

# 安装

script 引入 /dist/helper.umd.min.js
script 标签引入后会挂载 window.helper

npm i --save jason-helper

import {
  /* 各个方法去文档中找吧 */
} from 'jason-helper'
