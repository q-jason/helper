# 说明

!!! 该函数仅游览器端生效 !!!
主要作用为补足 h5 input 标签 type 为 number 时的 min max 功能
默认 type number 并设置 min 和 max 时，输入时还是会超出指定范围
本函数逻辑为若小于 min 或大于 max，则自动重置值为 min 和 max
